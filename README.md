# Handoff by CAIO GROUP

**A free Claude skill that moves any conversation to a new chat — with zero re-explanation.**

Built by [CAIO Group](https://wearecaio.com) — AI Executive Training & Consulting.

---

## What it does

When you're deep in a Claude session and need a fresh chat, type `/handoff`.

The skill automatically generates a structured `.md` file containing:

- **What you were working on** — session type, objective, approach
- **Decisions made** — every key choice from the conversation
- **Artifact classification** — all files, code, and drafts tagged as Carry Over, Reference, or Archive
- **Suggested chat name** — so your conversation history stays organized
- **Resume Prompt** — a paste-ready paragraph. Drop it as your first message in the new chat and Claude picks up immediately.

No re-explaining. No lost context. No starting from zero.

---

## Trigger phrases

The skill fires automatically on any of these:

- `/handoff`
- "move to a new chat"
- "move this to a new chat"
- "continue in a new chat"
- "handoff file"

---

## Installation

1. Download `handoff-by-caio.skill`
2. Place it in your Claude Cowork skills folder:

**Mac:**

```
~/Library/Application Support/Claude Cowork/skills/user/
```

**Windows:**

```
%APPDATA%\Claude Cowork\skills\user\
```

3. Restart Claude Cowork
4. Type `/handoff` in any conversation

---

## Output

The skill generates a clean `.md` file. No CAIO branding in the output — the file belongs to you entirely.
